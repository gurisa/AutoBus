Installation
###################

1. Create database named as you want (default: autobus), with default databse privileged (root:)
2. If you setup another database name or credential, then you need to setting up it on application/config/database.php configuration files.
3. Import database file (.sql) from database/autobus.sql
4. Access your domain (ex: localhost or another domain you've point up)
5. Well done!
